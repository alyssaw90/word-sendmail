# word-sendmail

To run the sample add-in code under node: 

1. git clone https://github.com/tomjebo/word-sendmail.git

2. cd word-sendmail\AdatumWordWeb

3. npm install

4. npm start

You should see something like this output:

~~~
$ npm start
